# cl1p-cli
> Cli tool for http://cl1p.net

## Installation
```sh
	$ yarn global add cl1p-cli

	# or if you'd like to use npm

	$ npm i -g cl1p-cli
```

## Usage
```sh
	Cli tool for http://cl1p.net

	Usage
	$ cl1p [text] --name <name>

	Options
	-n --name <name>                      # Cl1p name (cl1p.net/:name)
	-f --file <path>                      # Get clip content from file
	--lifetime <time>                     # Destroy the clip after given ms
```
